# Brevo Integration Reference

This folder contains project-specific notes for integrating FileMaker with Brevo.

The goal is not to mirror the full Brevo documentation. Use these files to record the current integration decisions, endpoint summaries, payload shapes, FileMaker architecture, and implementation notes.

## Official Docs

- Brevo API overview: https://developers.brevo.com/
- Transactional email guide: https://developers.brevo.com/docs/send-a-transactional-email
- Send transactional email endpoint: https://developers.brevo.com/reference/sendtransacemail

## Local Reference Files

- `transactional-email.md` - sending transactional email through the Brevo API
- `templates.md` - Brevo-hosted template strategy and FileMaker mapping
- `webhooks.md` - future inbound event handling
- `filemaker-architecture.md` - proposed FileMaker-side architecture
